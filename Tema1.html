JavaScript methods

JavaScript methods are actions that can be performed on objects.

A JavaScript method is a property containing a function definition.

Methods are functions stored as object properties.

Accessing Object Methods

You access an object method with the following syntax: objectName.methodName()

You will typically describe numeComplet as a method of the person object, and numeComplet as a property.

The numeComplet property will execute (as a function) when it is invoked with ().

This example accesses the numeComplet method of a person object:

name = person.numeComplet();

If you access the numeComplet property, without (), it will return the function definition:

name = person.numeComplet;

Adding a Method to an Object

Adding a new method to an object is easy:

person.name = function () {

return this.nume + " " + this.prenume;

};

What is this Keyword

Which object depends on how this is being invoked (used or called).

The this keyword refers to different objects depending on how it is used:

this is not a variable. It is a keyword. You cannot change the value of this.
